What is a List?

A list is a collection of items in some type T

Fixed Array

The most intituive way of implementing a list is to store the values of type T into a fixed size array

To insert an item, use array_name[index(start from 0)] = value.

Dynamic Array

Dynamic Arrays allows resizing of the original array. The ADT specifies to make a new array with the new size and copy the content from the old array, old array is then destoryed to free the memory.
