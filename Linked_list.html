Singly Linked List

List can be implemented by Linked list, each item in a linked list compose of the item with type T, and a pointer to the next element in the lined list.

During initilization, the start of the linked list is the head, and the head points to nullptr if there is nothing else in the list. After adding each elemnts, each elements added previously points to the current item, and current item points to the nullptr, signifies the end of the list.

Doubly Linked List

Singly linked list can easily be expanded and doesn't need to hassle to resize like dynamic arrays; however, it is very slow and inefficent to get the last item in the singly linked list since it is O(N) since you literally have to go through every item in the list to get the last item. Doubly linked list however, with a head and a tail, can easily solve this problem.

Notice that both the head and the tails of Doubly Linked list points to a nullptr to signify the end of the list.
